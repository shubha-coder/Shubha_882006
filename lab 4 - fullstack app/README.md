# Lab 4 - Fullstack App

This directory contains the final project for the full-stack React and Node.js application.

## Structure
- `/backend`: The Express server codebase holding the REST APIs for the counter and task manager.
- `/frontend`: The React codebase (built with Vite) that interfaces with the APIs and features modern Glassmorphism UI.

## How to Run

### Backend
Navigate to the backend folder and start the server:
`cd backend`
`node server.js`

### Frontend
Navigate to the frontend folder and spin up the Vite development server:
`cd frontend`
`npm run dev`
